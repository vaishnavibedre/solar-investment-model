# Solar Investment Financial Model

Utility-scale solar project financial evaluation using Python financial modelling and Power BI dashboarding.

## Project Objective

Evaluate financial viability of a 50 MW solar project using:

• NPV analysis  
• IRR modelling  
• Payback analysis  
• LCOE calculation  
• Scenario analysis  
• Sensitivity analysis  

## Tools Used

Python  
Pandas  
NumPy  
Matplotlib  
Power BI  

## Key Outputs

NPV: ₹560M  
IRR: 14.11%  
Payback: 7 years  
LCOE: ₹2.73/kWh  

## Dashboard

Power BI dashboard includes:

• Investment KPIs  
• Energy degradation profile  
• Cashflow payback visualization  
• Tariff scenario analysis  
• IRR sensitivity drivers

## Key Insights

• Project generates positive NPV and IRR above hurdle rate  
• LCOE below tariff confirms strong unit economics  
• Tariff is primary sensitivity driver  
• Project remains viable under downside scenarios
